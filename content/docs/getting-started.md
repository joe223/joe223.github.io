AriaType is a local-first voice keyboard for macOS. Here's how to set it up and start using it in your daily workflow.

## Installation

Download from [ariatype.com](https://ariatype.com) or check the source on [GitHub](https://github.com/joe223/AriaType). Install the app, and grant the required permissions. That's it—no account needed.

```bash
# Example: checking the version
ariatype --version
```

## Basic Usage

1. Hold `Cmd+/` for raw dictation
2. Hold `Opt+/` for polished output
3. Speak naturally, release when done

> Your voice never leaves your device. All processing happens locally.

### Supported Features

- **Fast transcription** - under 500ms average
- **CJK-friendly** - optimized for Chinese/Japanese/Korean
- Smart polish - removes fillers automatically

## Why Local-First?

| Feature | Local | Cloud |
|---------|-------|-------|
| Privacy | Full | Data sent |
| Speed | Fast | Depends |
| Cost | Free | Subscription |

Your voice never leaves your device. This means:

- No cloud dependency
- No subscription fees
- Complete privacy

*Works offline too.*
